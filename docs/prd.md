# 需求文档

## 1. 应用概述

### 1.1 应用名称
从政之路（政途人生）Mobile App v3.0.0

### 1.2 应用描述
写实向仕途模拟手机游戏，玩家从基层公职起步，通过四条职业线（行政线/党务线/纪检线/群团线）逐级晋升至最高领导层。v3.0.0版本进行职业线架构大改版：四条职业线从级别1~11，每级3~4个职位，每职位管辖4~5个部门，每部门有独立资金消耗与专属玩法页面，每职位配置专属考核指标，综合得分按加权计算影响晋升。游戏连接Supabase后端，提供完整从政生涯模拟体验。保留v2.8.0所有高级功能（双规审查/以权谋私/重大议案/历史评价/接班人/智库顾问团/导师计划/历史机遇/修宪提案等）。

## 2. 用户与使用场景

### 2.1 目标用户
- 对政治体制感兴趣的玩家
- 希望体验公务员晋升路径的用户
- 喜欢策略模拟类游戏的玩家

### 2.2 核心使用场景
- 用户通过手机号验证码完成注册登录
- 用户在建档系统完成角色创建，选择学历背景、从政动机、性格特征、家庭背景、初始职业线（行政线/党务线/纪检线/群团线）
- 用户从级别1起步，在主界面仪表盘查看当前职位、级别、职业线标识、综合考核得分
- 用户进入职位主界面，查看当前职位管辖的4~5个部门列表、预算状态
- 用户进入部门专属玩法页面，执行部门专属行动，消耗AP，影响部门KPI与资金消耗
- 用户在考核结果页面查看职位专属考核指标、各部门得分、加权综合得分
- 用户达到晋升条件时，进入完整六阶段晋升流程（民主推荐→组织考察→联审→常委票决→公示→任命+试用期）
- 用户在5个晋升节点可选择跨线转职，副厅级后线路永久锁定
- 用户体验完整从政生涯，直至退休或触发游戏结局

- 提供进入职位主界面按钮
- 提供进入考核结果页面按钮

- 提供后备干部池入口（资历达标但无空缺时显示）
- 提供双规预警入口（当isUnderInvestigation=true时显示）
- 提供特殊渠道入口（当级别>=3时显示）
- 提供重大议案入口（当级别>=人大代表级别且处于两会期间时显示）
- 提供历史评价入口（当距离退休<=3年时显示）
- 提供接班人培养入口（当级别>=12时显示）
- 提供卸任时机入口（当级别>=12且满足退休条件时显示）
- 提供智库顾问团入口（当级别>=9时显示）
- 提供导师计划入口（当级别>=10时显示）
- 提供百年一遇历史机遇入口（当级别>=11且触发历史机遇时显示）
- 提供修宪提案入口（当级别>=13时显示）

#### 3.2.5 职业线入口页面（career-path.tsx）
- 显示四条职业线选项卡（行政线/党务线/纪检线/群团线）
- 切换选项卡显示对应职业线的级别1~11结构树
- 每个级别显示3~4个职位名称
- 点击职位进入职位主界面
- 显示路线特权说明
- UI风格：按职业线区分颜色（行政线蓝灰/党务线正红/纪检线深蓝黑/群团线青春绿）

#### 3.2.6 职位主界面（position-hub.tsx）
- 显示职位名称、级别、职业线标识
- 显示职位专属考核指标列表（KPI名称、当前值、目标值、权重）
- 显示管辖部门列表（4~5个部门）
- 每个部门显示：部门名称、资金消耗系数、当前月度消耗、进入部门玩法页面按钮
- 显示上级专属拨款总额与剩余预算
- 显示预算状态（剩余预算、预算使用率、超支预警）
- 提供返回职业线入口页面按钮

#### 3.2.7 部门玩法页面（position-dept.tsx，数据驱动复用）
- 接收参数：line（职业线）、level（级别）、positionIndex（职位索引）、deptIndex（部门索引）
- 显示部门名称、所属职位、职业线标识
- 显示部门专属行动列表（2~4个行动，行动名称、AP消耗、冷却时间、效果描述）
- 玩家选择行动，消耗AP，执行行动
- 行动结果影响部门KPI、资金消耗、职位综合得分
- 显示部门当前KPI数值与资金消耗明细
- 提供返回职位主界面按钮

#### 3.2.8 职位专属KPI考核页面（position-kpi.tsx）
- 显示职位专属考核指标列表（4~5项指标，KPI名称、当前值、目标值、完成率、权重、得分）
- 显示各部门得分明细
- 显示加权综合得分计算公式与最终得分
- 显示考核档次（优秀/称职/基本称职/不称职）
- 显示考核结果对晋升的影响

#### 3.2.9 晋升系统页面（promotion.tsx，六阶段流程）
- 显示当前级别与职位
- 显示晋升前置硬性门槛校验结果：资历年限、政治条件、编制职数
- 显示晋升流程六阶段：
  + 阶段1：民主推荐（党委组织分层民主投票打分，得票前2名进入考察名单，可动用人脉拉票）
  + 阶段2：组织部组织考察（2~3人考察组谈话、查阅资料、走访，考察结论分为优秀/合格/暂缓使用/不宜提拔）
  + 阶段3：多部门联审+纪委廉政审查（纪委监委出具廉政意见，同步征求公安、信访、审计、网信部门意见）
  + 阶段4：常委会集体票决（党委常委会无记名投票，赞成票过半通过）
  + 阶段5：任前公示（公示5个工作日，实名举报暂停任命，网络舆情发酵撤销拟任）
  + 阶段6：正式任命+一年试用期（下发任职文件，试用期考核不合格降回原职级）
- 显示同级别其他职位列表（平级调动选项）
- 显示更高级别可选职位列表（晋升选项）
- 玩家选择目标职位，进入晋升流程
- 显示红色警告：确认晋升新岗位，原有岗位专属职权与事件永久清空，仅保留职级基础权限，无法回滚

#### 3.2.10 跨线转职页面

### 4.1 四条职业线架构（v3.0.0核心改版）

#### 4.1.1 职业线与级别结构
- 四条职业线：行政线/党务线/纪检线/群团线
- 每条职业线包含级别1~11
- 每个级别包含3~4个职位
- 每个职位管辖4~5个部门
- 四条线路完全独立，非晋升窗口期禁止跨线任职、禁止跨线兼任职务

#### 4.1.2 行政线职位体系（级别1-11）
- 级别1（科员）：乡镇科员、社区工作员、乡镇办事员
- 级别2（副科）：副镇长、乡镇办公室主任、民政助理
- 级别3（正科）：镇长、乡镇党委委员、科室主任、副乡镇党委书记
- 级别4（副处）：县委常委、副县长、政府办副主任、开发区副主任
- 级别5（正处）：县长/区长、县政府秘书长、开发区主任、县直局局长
- 级别6（正处上）：县委书记/区委书记、市直局局长、市政府助理调研员、县政协主席
- 级别7（副厅）：副市长、市政府秘书长、市直部门正职、驻外经济联络主任
- 级别8（正厅）：市长、市人大副主任、大型国企董事长、市直重点部门一把手
- 级别9（正厅上）：市委书记、市人大主任、市政协主席、市委政法委书记
- 级别10（副部）：副省长、省政府秘书长、省直厅局局长、国家部委副司长
- 级别11（正部）：省长/省委副书记、省人大主任、省政协主席、国家部委司长

- 级别5：县委组织部部长、县委宣传部部长、统战部长、政法委副书记
- 级别6：县委副书记、县委常委（党务）、政法委书记、县委巡察办主任
- 级别7：市委组织部部长、市委宣传部部长、市委统战部部长、市直机关工委书记

#### 4.1.6 职位与部门配置
- 职位配置数据结构：职位ID、职位名称、级别、职业线、专属考核指标列表、管辖部门列表、上级拨款总额
- 部门配置数据结构：部门ID、部门名称、所属职位ID、资金消耗系数、专属行动列表、KPI指标列表
- 考核指标配置数据结构：指标ID、指标名称、目标值、权重、计算公式

#### 4.1.7 数据驱动设计
- 职位主界面与部门玩法页面采用统一复用组件
- 页面根据职位ID/部门ID从配置数据中动态加载内容
- 避免硬编码数百个页面

#### 4.1.8 岗位玩法迭代机制
- 每一次晋升新职务，系统自动封存上一岗位全部专属事件、特殊职权与剧情玩法，仅保留对应职级通用永久权限
- 新岗位解锁全新独立内容
- 岗位专属内容一经封存无法回滚恢复
- 职级通用权限永久存档；岗位专属内容绑定单一职务，离任即归档锁死

### 4.2 考核系统

#### 4.2.1 职位专属考核指标
- 每个职位配置4~5个专属考核指标（KPI）
- 每个指标包含：指标名称、当前值、目标值、权重
- 示例：行政线级别3某职位考核指标：城市GDP增长率（权重30%）、财政收入完成率（权重25%）、民生满意度（权重20%）、重大项目完成度（权重15%）、安全生产事故率（权重10%）

#### 4.2.2 综合得分计算
- 综合得分=Σ（指标完成率×指标权重）
- 指标完成率=当前值/目标值×100%
- 综合得分影响考核档次：优秀（>=90分）、称职（>=75分）、基本称职（>=60分）、不称职（<60分）

#### 4.2.3 考核结果影响
- 综合得分达标是晋升的必要条件
- 考核档次影响晋升速度与职位选择范围

### 4.3 资金系统

#### 4.3.1 上级专属拨款
- 每个职位配置上级专属拨款总额
- 拨款总额按级别与职位区分（级别越高拨款越多，不同职位拨款不同）
- 示例：行政线级别3某职位年度拨款5000万元，党务线级别3某职位年度拨款3000万元

#### 4.3.2 部门资金消耗
- 每个部门配置资金消耗系数（按部门和级别区分）
- 部门月度消耗=基础消耗×消耗系数×部门活跃度
- 部门活跃度由玩家执行部门行动频率决定
- 示例：行政线级别3某职位下辖城建部门，消耗系数1.5，基础消耗100万元/月，活跃度1.2，则月度消耗=100×1.5×1.2=180万元

#### 4.3.3 预算管理
- 剩余预算=上级拨款总额-Σ各部门累计消耗
- 剩余预算<0时触发资金超支预警，影响考核得分
- 玩家需平衡各部门资金消耗，避免超支

### 4.4 部门玩法系统

#### 4.4.1 部门专属行动
- 每个部门配置2~4个专属行动
- 每个行动包含：行动名称、AP消耗、冷却时间、效果描述、影响的KPI指标、资金消耗增量
- 示例：行政线级别3某职位下辖城建部门，专属行动「审批重大工程项目」，AP消耗5，冷却时间3天，效果「城市GDP+2%，重大项目完成度+10%」，资金消耗增量50万元

#### 4.4.2 行动执行流程
- 玩家进入部门玩法页面，选择行动
- 系统检查AP是否充足，检查剩余预算是否充足，检查冷却时间是否结束
- 执行行动，消耗AP，增加部门资金消耗，更新部门KPI与职位综合得分，启动冷却时间
- 显示行动结果提示

### 4.5 晋升系统

#### 4.5.1 晋升前置硬性门槛（系统全自动校验，缺一不可）

##### 4.5.1.1 资历年限门槛
- 科员→副科：在岗满3年，至少2次年度考核结果为称职及以上
- 副科→正科：现任职务满3年，无任何党纪处分记录
- 正科→副处：正科级任职满3年，拥有至少一轮基层主官任职经历
- 副处→正处：副处级任职满3年，无年度考核不合格记录
- 正处→副厅：正处级任职满4年，具备两段及以上区县或市直单位主官履历
- 副厅→正厅：任职满4年
- 正厅→副部：任职满5年
- 副部→正部：任职满5年
- 副部级、正部级晋升高阶岗位：必须拥有跨省多地区岗位历练履历，系统自动检索人事档案库进行核验

##### 4.5.1.2 政治条件筛查（后台自动核验）
- 廉政档案干净：无纪委立案线索、无信访查实举报、离任审计无问题
- 民主测评达标：处级岗位提拔满意率>=75%，厅级岗位提拔满意率>=80%
- 底线问责约束：任期内未发生安全生产事故、群体性事件、重大舆情、环保问责。一旦被问责，晋升通道冻结1~3届
- 人事档案合规：学籍、工龄、党员履历材料完整无涂改。组织部档案会审不通过，直接暂停全部提拔流程

##### 4.5.1.3 编制职数壁垒
- 各级领导岗位设有固定编制职数
- 资历全部达标但本单位职数已满，自动进入后备干部池排队，等待人员退休、调动腾出空缺
- 组织部定期更新岗位空缺清单；无空缺时不会弹出晋升窗口，玩家只能调动至其他有空编的单位

#### 4.5.2 晋升流程六阶段

##### 4.5.2.1 阶段1：民主推荐（首轮筛选）
- 党委组织分层民主投票打分
- 得票前2名进入组织考察名单，其余候选人直接淘汰
- 可动用人脉拉票，过度操作会留下负面台账记录

##### 4.5.2.2 阶段2：组织部组织考察
- 2~3人考察组开展谈话、查阅资料、走访服务对象
- 系统随机确定谈话人员，玩家可动用关系微调人选，引导考察评价
- 考察结论分为四档：优秀、合格、暂缓使用、不宜提拔
- 暂缓使用：本次提拔搁置，保留下一届参选资格
- 不宜提拔：晋升资格冻结两届

##### 4.5.2.3 阶段3：多部门联审+纪委廉政审查
- 纪委监委出具廉政意见，存在案件线索直接一票否决。小额轻微违纪可协调处置
- 同步征求公安、信访、审计、网信部门意见，任一单位出具负面意见，终止提拔程序

##### 4.5.2.4 阶段4：常委会集体票决
- 党委常委会无记名投票
- 赞成票过半方可通过，派系对抗会出现常委投反对票

##### 4.5.2.5 阶段5：任前公示
- 常规公示时长5个工作日
- 实名有效举报必须暂停任命并重新核查；匿名举报可酌情处置
- 网络舆情大面积发酵，直接撤销拟任决定

##### 4.5.2.6 阶段6：正式任命+一年试用期
- 公示无异议后下发任职文件
- 新任领导干部全部设置一年试用期
- 试用期考核不合格，降回原职级；考核合格正式定岗

#### 4.5.3 职位选择
- 玩家可选择同级别其他职位（平级调动）
- 玩家可选择更高级别职位（晋升）
- 不同职位管辖不同部门，玩法完全不同

#### 4.5.4 晋升弹窗交互
- 显示「职务空缺通知」
- 提供选项：
  + 选项1：参与本次岗位竞争（启动民主推荐流程，进入派系博弈）
  + 选项2：放弃本次名额，进入后备干部库，等待下一届空编
  + 选项3：调动至其他有空编单位
  + 选项4（副厅级之前可见）：申请干部跨条线交流，切换发展赛道
- 显示红色警告：
  + 确认晋升新岗位，原有岗位专属职权与事件永久清空，仅保留职级基础权限，无法回滚
  + 跨线路转职仅剩余X次机会，晋升副厅级后交流通道永久关闭
  + 本次任命存在一年试用期，试用考核不达标将被免去新职务

### 4.6 跨线转职交流机制

#### 4.6.1 转职窗口期严格锁定
- 仅能在五次晋升节点切换赛道：科员→副科、副科→正科、正科→副处、副处→正处、正处→副厅
- 晋升副厅级后线路永久锁定，档案划入省管干部序列，终身无法更换赛道

#### 4.6.2 转职难度梯度
- 跨线交流需要上级组织部干部交流审批，无法一键自由切换线路
- 转职难度：纪检线转出＞行政/党务互转＞群团转入行政、党务
- 纪检线向外转职，必须额外补齐基层挂职履历
- 每次转职都会重新核算基层工作年限，履历短板会拉长后续晋升周期
- 转职完成后，原线路履历封存归档，新岗位从零开始积累考核年限

#### 4.6.3 兼任禁令
- 系统后台自动拦截跨线路兼职申请，不允许同时兼任两条线路职务

### 4.7 后备干部排队机制
- 资历达标但无岗位空缺，自动划入后备人才库，分为三梯队：
  + 第一梯队（近期提拔）：有空岗优先补位
  + 第二梯队（中期储备）：最多等候两届任期
  + 第三梯队（长期储备）：必须补齐基层履历才能参与竞争
- 组织部从后备库里择优调任填补空缺岗位
- 长期闲置得不到任用，积累消沉值，下一轮民主测评得分降低

### 4.8 破格提拔规则（高阶权限）
- 常规提拔严格卡死任职年限；破格提拔需要至少满足2项实绩条件：
  + 重大招商引资项目落地
  + 处置突发事件立功受奖
  + 获评省级及以上个人表彰
  + 在乡村振兴、维稳一线连续担任主官满规定年限
- 破格提拔需要上级组织部专项审批，公示延长至15天，极易遭到对手实名举报，廉政风险大幅上升
- 副厅级及以上岗位取消破格权限，必须逐级满年限晋升

### 4.9 处分、问责与降职冻结机制
- 年度考核不合格：职级保留，晋升冻结一届
- 党内警告处分：晋升冻结两届；严重警告处分：晋升冻结三届
- 履职失败、项目烂尾、舆情失控：免去领导职务，转为同级非领导职务
- 离任审计查出财务问题：永久终止提拔资格，转入纪委案件核查程序
- 全程禁止跨线路兼任职务，系统程序自动拦截违规任命

### 4.10 AP系统
- 每日行动体力（AP）系统，每天恢复
- 执行行动消耗AP
- AP不足时无法执行行动

### 4.11 随机事件
- 随机触发事件，提供3个选项
- 每选项有不同效果组合
- 玩家选择处理方式，影响数值与晋升进度

### 4.12 上级关系
- 直属上司好感值影响晋升推荐
- 提供汇报工作、请示问题、节日问候等行动

### 4.13 人脉关系网络
- 六类人脉：同学关系、同僚关系、商界人脉、学术资源、媒体关系、中央/省级人脉
- 人脉关系值影响各类行动效果

### 4.14 派系政治
- 三大派系：改革派/务实派/保守派
- 玩家选择跟随某派系大佬或保持独立
- 每次重要决策影响各派系倾向值

### 4.15 双规审查系统
- 触发条件：isUnderInvestigation=true
- 6类证据：受贿证据/财产申报异常/违规审批记录/通讯记录/证人证词/生活作风问题线索
- 玩家选择坦白认罪/辩护抵赖/选择辩护律师
- 辩护成功回归正常，失败游戏结束

### 4.16 以权谋私玩法
- 解锁条件：级别>=3
- 4类行动：项目批复回扣/土地出让暗利/违规审批开绿灯/安插关系户就业
- 每次操作增加巡视风险值

### 4.17 重大议案系统
- 解锁条件：级别>=人大代表级别
- 触发时机：每届（5年）两会期间
- 可提交1-3个议案，消耗政治资本
- 拉拢代表票数，通过的议案解锁新政策工具

### 4.18 历史评价系统
- 解锁条件：距离退休<=3年
- 四维度评分：经济维度/民生维度/廉洁维度/改革维度
- 历史定性：改革先行者/稳健实干家/平庸守成者

### 4.19 接班人路线图
- 解锁条件：级别>=12
- 选拔1名接班人，持续投入关注值与历练派遣
- 接位成功获得「政治遗产」加成分

### 4.20 卸任时机选择
- 解锁条件：级别>=12且满足退休条件
- 主动退休：锁定当前历史评价得分
- 强行续任：风险系数+50%，一旦触发重大负面事件，历史评价全盘清零

### 4.21 智库顾问团
- 解锁条件：级别>=9
- 从科学界/经济学界/法学界各招募1名顾问
- 政策制定时提供学术背书

### 4.22 导师计划
- 解锁条件：级别>=10
- 每届选3名潜力股重点指导
- 指导成功后成为长期政治资产

### 4.23 百年一遇历史机遇
- 解锁条件：级别>=11
- 随机触发「百年一遇」历史机遇
- 成功则永久历史档案记录+最高政绩成就

### 4.24 修宪提案
- 解锁条件：级别>=13
- 选择推动修宪提案或维护现状
- 推动修宪需极高政治资本，成功历史级成就

### 4.25 游戏时间与日历
- 游戏内时间流逝，每次行动推进天数
- 重要节日事件：春节/国庆/党代会等
- 5年一次换届节点
- 65岁退休机制

### 4.26 秘书处系统（secretary.tsx，重构+新增）

#### 4.26.1 秘书成长系统
- 秘书经验值：每次操作+1-5经验
- 秘书等级：初级（0-99经验）/助理（100-299经验）/主任（300-599经验）/资深（600-999经验）/首席（1000+经验）
- 每级解锁新文字描述
- 经验值存储在player_saves的secretary字段中

#### 4.26.2 文件批示系统
- 每次进入页面随机生成3-5份待批示文件
- 文件类型：请示/报告/方案/建议
- 文件分类：经济类/民生类/生态类/党建类/安全类
- 处理方式与效果：
  + 批准：按文件效果执行（消耗能力，获得政绩+属性收益）
  + 修改：效果减半但廉洁+1
  + 驳回：获少量政绩，无属性变化
  + 搁置：无任何效果
- 文件数据通过本地state模拟，不存储数据库

#### 4.26.3 舆情管理系统（rank4+解锁）
- 每月随机生成1-3条舆情事件
- 舆情类型：负面/中性/正面
- 正面舆情：公示宣传，获得额外政绩+上司好感
- 负面舆情处理方式：
  + 删除：快速降低热度，廉洁-1
  + 澄清：热度降低较慢，稳定性+1
  + 沉默：热度持续上升，无属性变化
  + 危机公关：热度大幅降低，消耗政绩
- 舆情有时效（超期不处理加倍扣分）
- 舆情热度指数可视化展示
- 舆情数据通过本地state模拟，不存储数据库

#### 4.26.4 公务协调系统（整合并扩展）
- 联络上级部门（rank3+）：提升上司好感，消耗AP
- 协调平级单位（rank3+）：提升同僚关系，消耗AP
- 接待来访领导（rank7+）：显示来访领导信息、接待方案选择、接待结果，影响人脉与政绩
- 跨省对接（rank10+）：提升中央/省级人脉，消耗AP
- 干部协调（推荐任职）：推荐下属晋升，消耗政治资本
- 巡查督导：检查下属单位，获得政绩

#### 4.26.5 辅助功能系统（保留并重构）
- 待办提醒：显示待办事项列表
- 报告摘要：显示各部门报告摘要
- 起草文件：生成文件草稿，消耗AP
- 辅助申请：协助提交各类申请
- 安保配置：配置安保方案
- 智能开关：自动施政/自动招募开关
- 整理公文：消耗AP，换取政绩
- 情报汇报：获取情报信息
- 决策参谋：提供决策建议

| 用户综合考核得分不达标时申请晋升 | 弹窗提示「综合考核得分不达标，无法晋升」 |

| 用户晋升副厅级后申请跨线转职 | 弹窗提示「已晋升副厅级，线路永久锁定，无法跨线转职」 |
| 用户尝试跨线兼任职务 | 系统后台自动拦截，弹窗提示「禁止跨线路兼任职务」 |
| 用户在部门行动冷却时间内再次执行该行动 | 弹窗提示「该行动冷却中，剩余X天」 |
| 用户在民主推荐阶段得票未进入前2名 | 弹窗提示「民主推荐未通过，本次晋升失败」 |
| 用户在组织考察阶段结论为「暂缓使用」 | 弹窗提示「组织考察结论为暂缓使用，本次提拔搁置，保留下一届参选资格」 |

| 用户在常委会票决阶段赞成票未过半 | 弹窗提示「常委会票决未通过，本次晋升失败」 |

| 用户试用期考核不合格 | 弹窗提示「试用期考核不合格，降回原职级」 |
| 用户长期在后备干部池闲置 | 积累消沉值，下一轮民主测评得分降低，弹窗提示「长期未获任用，民主测评得分下降」 |
| 用户rank<4时进入舆情管理Tab | 弹窗提示「舆情管理系统需rank4+解锁」 |
| 用户rank<3时执行联络上级部门/协调平级单位 | 弹窗提示「该功能需rank3+解锁」 |
| 用户rank<7时执行接待来访领导 | 弹窗提示「该功能需rank7+解锁」 |
| 用户rank<10时执行跨省对接 | 弹窗提示「该功能需rank10+解锁」 |
| 用户在文件批示Tab中选择批准但能力不足 | 弹窗提示「能力不足，无法批准该文件」 |

| 网络异常导致存档保存失败 | 弹窗提示「操作失败，数据未保存」，不改变游戏状态 |
| 用户快速连点提交按钮 | 300ms防抖机制，避免重复提交 |

## 7. 验收标准

1. 用户完成手机号验证码注册登录
2. 用户在建档系统完成角色创建，选择初始职业线（行政线/党务线/纪检线/群团线）
3. 用户从级别1起步，在主界面仪表盘查看当前职位、级别、职业线标识、综合考核得分
4. 用户进入职业线入口页面，查看四条职业线的级别1~11结构树，选择职位进入职位主界面
5. 用户在职位主界面查看职位专属考核指标、管辖部门列表、上级拨款与剩余预算
6. 用户进入部门玩法页面，执行部门专属行动，消耗AP，影响部门KPI与资金消耗
7. 用户在考核结果页面查看各部门得分、加权综合得分、考核档次
8. 用户达到晋升条件时，进入完整六阶段晋升流程（民主推荐→组织考察→联审→常委票决→公示→任命+试用期），完成晋升
9. 用户在5个晋升节点选择跨线转职，提交上级组织部审批，完成转职
10. 用户进入秘书处页面，查看秘书信息与经验值，执行文件批示、舆情管理、公务协调、辅助功能等操作
11. 用户体验完整从政生涯，直至退休或触发游戏结局
